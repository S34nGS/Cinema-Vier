# Cinema Project
Hoe run je de code?

## Database updaten / aanmaken
run het CreateDB.bat of CreateDB.sh script (bat voor windows, voor mac / linux de sh)

## Linux / Macos
voer het volgende bestand uit in je terminal, in de root directory van dit project
```
./run.sh
```

## Windows
voer het volgende bestand uit in je terminal, in de root directory van dit project
```
./run.bat
```
